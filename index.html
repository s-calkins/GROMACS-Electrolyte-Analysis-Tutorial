GROMACS Electrolytic Solution Tutorial

Tutorial Overview

This GROMACS tutorial details the modeling and analysis of an aqueous zinc sulfate (ZnSO₄) electrolyte system. The tutorial will begin with environment configuration using Packmol, and will then detail processing and data analysis, consisting of a brief investigation of solvation structures through calculating radial distribution functions (RDFs), and various diffusion coefficients through computing the mean squared displacement (MSD) of the ionic species.

This tutorial applies to all GROMACS versions within the 2018.x series.

Section 1 : Modeling the System

(1.1) Encoding Molecules

To create a solution using Packmol, the program will require files that detail specifics of each molecule used. In this case, that meas generating files for our solution's Water, Zinc, and Sulfate.

First, save Water as water.pdb with the following specifics: ATOM 1 OW SOL 1 0.000 0.000 0.000 1.00 0.00 O ATOM 2 HW1 SOL 1 0.096 0.000 0.000 1.00 0.00 H ATOM 3 HW2 SOL 1 -0.024 0.093 0.000 1.00 0.00 H TER END

Then, Zinc as zn.pdb: ATOM 1 ZN ZN2 1 0.000 0.000 0.000 1.00 0.00 Zn TER END

And Sulfate as so4.pdb: HETATM 1 S SO4 1 0.000 0.000 0.000 1.00 0.00 S HETATM 2 O1 SO4 1 1.430 0.000 0.000 1.00 0.00 O HETATM 3 O2 SO4 1 -0.477 1.365 0.000 1.00 0.00 O HETATM 4 O3 SO4 1 -0.477 -0.682 1.181 1.00 0.00 O HETATM 5 O4 SO4 1 -0.477 -0.682 -1.181 1.00 0.00 O TER END

(1.2) Create & Generate Using Packmol

Save the Packmol Input File as packmol.inp: tolerance 2.0 filetype pdb output system_raw.pdb # box dimensions in Å (4 nm = 40 Å) add_box_sides 0.0 structure water.pdb number 4000 inside box 0. 0. 0. 40. 40. 40. end structure structure so4.pdb number 58 inside box 0. 0. 0. 40. 40. 40. end structure structure zn.pdb number 58 inside box 0. 0. 0. 40. 40. 40. end structure

Next, run Packmol in your simulation's main folder: packmol < packmol.inp The command will generate system_raw.pdb, a packed, roughly neutral 1.5M ZnSO4 (aq) box.
